In CSS, there are many properties and rulesets specific for dealing with typography. Here is a collection of the most useful ones:

*font-family*

You can use many different typographies in your website. The easiest and most common way is to use https://fonts.google.com/ You just need to select a font family from their database, and include the link inside your HTML's < head> tag! Then, in your CSS file you just have to write: font-family: 'Font Name here';

*font-weight*

You also have the option of choosing which font-weight to use, from the THINNEST one to the BOLDEST!

*font-size*

The font size can also be modified!
Pixel values (e.g. 40px) are easy to use but not great for page responsity
Relative values (e.g 4em, 4rem) adapt to the container and window size
